# 📈 Bitforce Live Ledger Engine

Welcome to the **Bitforce Live Ledger Engine** dashboard. This is a premium Streamlit-based web application designed to track, calculate, and optimize compounding returns across a multi-cycle investment lifecycle. It helps users visualising their actual growth vs. potential compounding milestones in real time.

## 🚀 Features

- **Dynamic Lifecycle Tracking:** Real-time logging of active capital investments across 7 custom cycles.
- **Smart Wallet Distribution:** Automated splitting of profit into Main Wallet Core (70%) and Support Wallet Rolling Base (30%).
- **Auto Round Re-investment:** Intelligent compounding logic that secures decimal fragments while auto-routing rounded capital amounts.
- **Compounding Opportunity Loss Report:** Interactive analytical warnings that dynamically calculate missed target impacts if capital is prematurely withdrawn.
- **Theoretical Compounding Map:** A fixed roadmap displaying 0% withdrawal projections from the initial setup value.
